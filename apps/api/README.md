# @samajh/api

Runnable Node.js API for Samajh Match MVP.

## Modules

- Auth
- Profile
- Match
- Chat
- Safety + Admin moderation

## Run

```bash
npm run start:api
```

## Test

```bash
npm run test
```

## Auth usage

Use bearer token from `POST /auth/login`:

```http
Authorization: Bearer <token>
```
